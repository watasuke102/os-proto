# os-proto

Self-made Operating System prototype for x86-64, written in Rust

## commands

```bash
# build
$ cargo xtask b
$ cargo xtask build

# run
$ cargo xtask r
$ cargo xtask run

# build + run
$ cargo xtask
$ cargo xtask a
$ cargo xtask all
```
